Curso de Sistemas de Informação

Introdução

O curso de Sistemas de Informação na Unifametro é voltado para a formação de profissionais capazes de analisar, projetar, implementar e gerenciar sistemas de informação que atendam às necessidades das organizações.

O que Aprendi de HTML

Durante as aulas práticas, aprendi a usar diversas tags HTML para estruturar e formatar o conteúdo de páginas web. Aqui estão algumas das tags que aprendi:

Cabeçalhos

Tags de cabeçalho são usadas para definir títulos e subtítulos:

  • Título Principal (h1)

  • Subtítulo (h2)

  • Subtítulo Menor (h3)

  • Seção (h4)

  • Subseção (h5)
  • Subseção Menor (h6)

Parágrafos e Formatação de Texto

Os parágrafos são definidos com a tag <p> e podemos usar várias tags para formatar o texto:

Exemplo de texto marcado, texto pequeno, itálico, sublinhado e negrito.

Listas

Aprendi a criar listas ordenadas e não ordenadas:

Lista Ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenada:

  • Item A
  • Item B
  • Item C

Links e Citações

Links são criados com a tag <a>. Por exemplo, Visite o site da Unifametro.

Esta é uma citação em bloco para destacar um texto importante ou uma citação.

Outras Tags Úteis

Além das tags básicas, aprendi algumas tags adicionais:

  • <hr> para criar uma linha horizontal.
  • <sub> para subscrito e <sup> para sobrescrito.
  • <abbr> para abreviações, como HTML.
  • <font> para mudar a cor e outros atributos do texto, por exemplo, texto azul.
  • <del> para indicar texto deletado, como este texto foi removido.
